# **MCES10 Software AI Dataset Utility**

### Features
- Format Data Efficiently and Effectively
- Easy to configure
- DMG Installer


### Compiling 
  Ensure you have the latest version of PyInstaller Installed
 - <code> git clone https://github.com/MCES10-Software/MCES10-Software-AI-Dataset-Utility.git </code>
 - <code> cd MCES10-Software-AI-Dataset-Utility </code>
 - <code> chmod u+x buildapps.sh </code>
 - <code> bash buildapps.sh </code>
 
Please note the app may take a while to load.
